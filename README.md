# Acquisition Management System Demo
 
## Setup Instructions

1. **Install Dependencies**
```bash
   npm install
```

2. **Environment Setup**
```bash
   cp .env.example .env
```

3. **Start Development Server**
```bash
   npm start
```

4. **Build for Production**
```bash
   npm run build
```

## Features
- Contract Management
- Vendor Management
- Dashboard Analytics
- Search & Filter
- Progress Tracking

## Tech Stack
- React 18
- Tailwind CSS
- Lucide React Icons
- React Router
